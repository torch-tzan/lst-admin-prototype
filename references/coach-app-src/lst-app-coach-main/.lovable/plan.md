

# 縮小日曆尺寸以符合截圖比例

## 分析

截圖中的日曆格子較小且緊湊，月份導航按鈕也較小。目前程式碼中：
- 日期格子 `w-10 h-10`（40px），按鈕 padding `py-[6px]`
- 月份導航按鈕 `w-9 h-9`
- 月份導航與日期間距 `mb-5`
- 字體 `text-[15px]`
- 選中圓角 `rounded-[6px]`

截圖中日期格更緊湊，選中日期是圓形（非圓角方形），整體行距更小。

## 計畫

修改 `src/pages/CalendarPage.tsx`：

1. **月份導航**：縮小按鈕 `w-8 h-8`，減少下方間距 `mb-3`
2. **星期標題**：字體縮小 `text-xs`，padding 縮小 `py-1`
3. **日期格子**：
   - 格子尺寸從 `w-10 h-10` 改為 `w-9 h-9`
   - 按鈕 padding 從 `py-[6px]` 改為 `py-[3px]`
   - 字體從 `text-[15px]` 改為 `text-[14px]`
   - 選中狀態圓角從 `rounded-[6px]` 改為 `rounded-full`（圓形，符合截圖）
4. **容器** padding 從 `pt-3 pb-3` 改為 `pt-2 pb-2`

這些調整會讓日曆整體更緊湊，接近截圖效果。

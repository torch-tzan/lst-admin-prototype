## 待客戶確認的模糊地帶

1. **場館 admin 能否看到本場館的教練匯款帳戶？**（目前預設不能，要 LST 統一管理）
2. **低分評價下架：場館能直接下架還是要 LST 審核？**（prototype 先做直接下架）
3. **場館能否發本場館公告？**（目前只 LST 能發；實務上場館可能要發本場館的營業通知）
4. **LST admin 能否代場館處理預約？**（緊急情況下的 override，目前未實作）
